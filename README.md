# Projet Fil Rouge 1 - Santé Publique

Ce projet vise à optimiser la répartition des ressources sanitaires en République du Congo.
Il inclut :
- Une base de données pour centraliser les données hospitalières.
- Des analyses exploratoires (Pandas, Power BI).
- Un modèle de prédiction de la fréquentation hospitalière.
- Un tableau de bord interactif pour les décideurss.

Auteur : Duciel Enguela
Formation : Akieni Academy - Cohorte 2
